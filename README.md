Тестовый проект по основам работы с базой данных "Игровая коллекция"

Структура базы данных

База данных состоит из следующих таблиц:

1. Таблица платформ :

   · id

   · название

   · компания-производитель

2. Таблица прогресса :

   · id

   · статус игры

   · сыграно часов

   · оценка

3. Таблица игр 

   · id

   · название игры

   · id платформы

   · жанр

   · год выпуска

   · id прогресса
